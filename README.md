## Dot files for my arch+i3 setup

to install, just...
```
chezmoi init --apply $GITHUB_USERNAME
```
in case I'll be using a private repo for some reason, run this instead...
```
chezmoi init --apply git@github.com:$GITHUB_USERNAME/dotfiles.git
```
